# 📢 패키지#02 인쇄 기능 추가 안내

안녕하세요. EdgeEnglishLab입니다.

이번 업데이트로 **패키지#02**의 인쇄 기능이 추가되었습니다. 생성한 문제를 종이로 출력하거나 파일로 저장하여 언제든지 활용하실 수 있습니다.

## ✨ 새로운 기능

### 1. 문제 페이지 인쇄
- 패키지#02로 생성한 문제를 인쇄용으로 출력할 수 있습니다
- A4 가로 형식으로 최적화되어 있어 깔끔하게 출력됩니다
- 모든 유형(#01~#11, #13, #14)의 문제가 포함됩니다

### 2. 정답 페이지 인쇄
- 문제와 함께 정답 페이지도 별도로 인쇄할 수 있습니다
- 각 문제의 정답과 해설이 포함되어 있습니다
- 영어 본문과 함께 한글 해석도 함께 확인할 수 있습니다

### 3. 파일 다운로드 지원
문제나 정답 페이지를 다음 형식으로 다운로드할 수 있습니다:
- **PDF 파일**: 인쇄하기 편한 PDF 형식으로 저장
- **DOC 파일**: Microsoft Word 문서 형식으로 저장

## 📋 사용 방법

### 인쇄(문제) 버튼 사용
1. 패키지#02로 문제를 생성합니다
2. 화면 하단의 **"인쇄(문제)"** 버튼을 클릭합니다
3. 파일 형식을 선택합니다 (PDF 또는 DOC)
4. 파일이 자동으로 다운로드됩니다

### 인쇄(정답) 버튼 사용
1. 패키지#02로 문제를 생성합니다
2. 화면 하단의 **"인쇄(정답)"** 버튼을 클릭합니다
3. 파일 형식을 선택합니다 (PDF 또는 DOC)
4. 정답이 포함된 파일이 자동으로 다운로드됩니다

## 💡 주요 특징

- **깔끔한 레이아웃**: A4 가로 형식으로 최적화된 레이아웃
- **모든 유형 지원**: 패키지#02의 모든 문제 유형이 인쇄 가능
- **정답 포함**: 정답 모드에서는 정답과 해설이 함께 표시됩니다
- **한글 해석**: 영어 본문의 한글 해석도 함께 확인 가능
- **파일 저장**: PDF나 Word 문서로 저장하여 보관하거나 공유할 수 있습니다

## 🎯 활용 팁

## 📝 참고사항

- 인쇄된 파일은 Firebase Storage에 자동으로 저장됩니다
- 파일명은 자동으로 생성되며 날짜와 시간이 포함됩니다
- PDF 형식은 브라우저 인쇄 기능을 사용합니다
